# graph_parameters_analysis

A graph algorithms assignment. The program reads graphs and computes a set of their parameters.

## PL

Program wczytuje grafy i wyznacza następujące parametry:

**Ciąg stopniowy** – stopnie wszystkich wierzchołków w kolejności ich numerów

**Liczba składowych spójności** – ile spójnych składowych zawiera graf

**Dwudzielność** – czy graf jest dwudzielny

**Mimośrodowość wierzchołków** – dla każdego wierzchołka, w ramach jego składowej spójności

**Kolorowanie wierzchołków** trzema metodami:

- zachłanna (wg numeru wierzchołka)
- LF – _Largest First_
- SLF – _Smallest Last First_

**Liczba krawędzi dopełnienia grafu**

## EN

The program reads graphs and computes the following parameters:

**Degree sequence** – degrees of all vertices in order of their indices

**Number of connected components**

**Bipartiteness** – whether the graph is bipartite

**Vertex eccentricity** – for each vertex, within its connected component

**Vertex colouring** using three methods:

- Greedy (by vertex number)
- LF – _Largest First_
- SLF – _Smallest Last First_

**Number of edges in the graph's complement**
